Security

Your Personal Access Token is stored securely in your operating system's keychain/credential manager. It is never stored in the browser or extension storage. The native messaging host (a local program) retrieves the token only when creating bookmarks.

Quick Start Guide

  1. Create a public GitHub repository for your bookmarks
  2. Generate a Personal Access Token with repo or public_repo scope
  3. Enter your GitHub username and repository name above
  4. Paste and save your PAT
  5. Click "Test Connection" to verify everything works
  6. Start bookmarking! Click the extension icon on any page
